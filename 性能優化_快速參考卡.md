## 🎯 四大瓶頸速查表

### 🔴 P0-1：Cloudinary Logo 延遲 (-800ms)
```
❌ 當前：
<img 
  src="https://res.cloudinary.com/dvizdsv4m/image/upload/...png"
  loading="eager"
/>

✅ 優化方案 A（推薦）：改 SVG
import { LOGO_SVG } from '@/assets'
<img src={LOGO_SVG} loading="lazy" />

✅ 優化方案 B：Cloudinary 參數優化
src="https://res.cloudinary.com/dvizdsv4m/image/upload/f_auto,q_auto:best,w_200,dpr_2/logo.png"
```

### 🔴 P0-2：LIFF 初始化阻塞 (-1000ms)
```
❌ 當前：
liff.init({ liffId })
  .then(() => setLiffObject(liff))
  // 卡住直到 LIFF 響應

✅ 優化方案：非阻塞 + Timeout
import { initLiffWithTimeout } from '@/utils/performance-optimization'

const success = await initLiffWithTimeout(liffId, 5000)
setLiffReady(true)  // ⭐ 立即設為 ready，不等 LIFF
```

### 🟠 P1-1：stampCount 每秒輪詢 (-定期卡頓)
```
❌ 當前：
useEffect(() => {
  const interval = setInterval(() => {
    setStampCount(getUnlockedStampCount())  // 每秒！
  }, 1000)
}, [])

✅ 優化方案：事件驅動
import { useStampCount } from '@/utils/performance-optimization'

const stampCount = useStampCount()  // 只在改變時更新
```

### 🟠 P1-2：GA4 同步追蹤 (-50-100ms)
```
❌ 當前：
<button onClick={() => {
  trackEvent('click', {...})  // 阻塞點擊反應
  handleClick()
}}>

✅ 優化方案：非同步隊列
import { analyticsQueue } from '@/utils/performance-optimization'

<button onClick={() => {
  analyticsQueue.push('click', {...})  // 非同步
  handleClick()
}}>
```

## 📌 核心代碼片段速查

### 代碼片段 1：非阻塞 LIFF
```tsx
// 直接複製到 LiffContext.tsx
import { initLiffWithTimeout } from '@/utils/performance-optimization'

useEffect(() => {
  const liffId = import.meta.env.VITE_LIFF_ID
  
  setLiffReady(true)  // 立即設為 ready
  
  // 後台初始化
  initLiffWithTimeout(liffId, 5000).then(success => {
    if (success && liff.isLoggedIn()) {
      liff.getProfile().then(profile => setProfile(profile))
    }
  })
}, [])
```

### 代碼片段 2：事件驅動 Stamp
```tsx
// 在 passportUtils.ts 中
export function emitStampUnlockedEvent(stampId: string) {
  document.dispatchEvent(new CustomEvent('stamp-unlocked', { detail: { stampId } }))
}

// 在需要的地方調用
unlockStamp(stampId)
emitStampUnlockedEvent(stampId)  // ⭐ 加這一行

// Header.tsx 中
const stampCount = useStampCount()
```

### 代碼片段 3：GA4 非同步
```tsx
// 全專案替換
❌ trackEvent('button_click', { name: 'unlock' })
✅ analyticsQueue.push('button_click', { name: 'unlock' })
```
