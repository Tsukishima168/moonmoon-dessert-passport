⚡ 快速開始 (跳過閱讀版):

  1. 前往 https://supabase.com
  
  2. 建立新 Project:
     - 名稱: moonmoon-passport
     - 選擇最近的地區
  
  3. 等待 Project 建立完成
  
  4. 點擊 Settings → API
  
  5. 複製 Project URL 和 Anon Key
  
  6. 建立 .env.local:
     VITE_SUPABASE_URL=your_url
     VITE_SUPABASE_ANON_KEY=your_key
     VITE_LIFF_ID=your_liff_id
  
  7. 在 Supabase SQL Editor 執行:
     docs/passport_schema.sql
  
  8. 執行命令:
     npm install @supabase/supabase-js
  
  9. 驗證: 在 Supabase 查看 5 個表是否已建立
